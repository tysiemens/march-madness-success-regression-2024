# NCAA March Madness Regression Analysis

R Markdown regression and correlation analysis on 2023-24 NCAA basketball stats during March Madness

---

## Overview

This project analyzes the linear regression of NCAA team stats on how far the team progressed in the March Madness Tournament. The goal is to identify the stats with the highest predictive value in tournament success and visualize the correlation 

---

## Features

- Multiple linear regression
- Multiple regression feature selection
- Correlational visualization
- Step-by-step explaination + discussion
  
---

## Dataset

Source: College Basketball Reference

Contains 36 team statistics from 362 D1 NCAA men's basketball teams from the 2023-24 season

Important variables:
- `School`
- `SRS`
- `SOS`
- `Conf.W`
- `Conf.L`
